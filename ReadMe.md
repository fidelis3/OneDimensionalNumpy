
# OneDimensionalNumpy

A repository for learning and practicing one-dimensional NumPy operations and array manipulations.

## Overview

This project contains examples and exercises focused on fundamental NumPy functionality for working with 1D arrays, including creation, indexing, slicing, and common operations.

## Features

- Array creation and initialization
- Indexing and slicing operations
- Mathematical operations
- Array manipulation and reshaping
- Statistical functions


### Installation

```bash
pip install numpy
```
